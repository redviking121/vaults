# 🟥 1. What schema_version _is_

**schema_version** is a **declared semantic version** embedded inside the entity itself.

It is always interpreted as:

```
MAJOR.MINOR.PATCH
```

- **MAJOR** — breaking shape changes
- **MINOR** — additive fields, new optional surfaces
- **PATCH** — corrections, clarifications, non‑structural fixes

This is the field that binds the entity to a specific **Version_signal** tier.

If schema_version exists, the Engine does **not** guess.
