The membrane evaluates three layers:

### 1. **Pattern‑of‑Presence**

Which fields appear together across snapshots.  
This forms a **co‑occurrence matrix**.

### 2. **Pattern‑of‑Shape**

How fields are structured:

- nested
- flat
- list‑based
- map‑based

This forms a **shape topology**.

### 3. **Pattern‑of‑Semantics**

Which fields carry meaning as a group:

- timestamps
- identifiers
- resource descriptors
- state clusters

This forms a **semantic motif**.

Together, these produce the **Field_pattern_signal vector** — the entity’s structural fingerprint.
