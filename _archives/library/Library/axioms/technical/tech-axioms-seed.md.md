# Technical Axioms — Seed Set (v0.1)
**Location:** /axioms/technical/tech-axioms-seed.md  
**Status:** Seed

## TECH‑01 — Every Object Has a Stable ID
IDs never change, even if names or locations do.

## TECH‑02 — Links Must Be Valid
Broken links are system errors, not cosmetic issues.

## TECH‑03 — No Cycles in Inference Graphs
Inference graphs must remain acyclic to preserve reasoning integrity.

## TECH‑04 — Metadata Is First‑Class
Metadata is not decoration; it is part of the knowledge object.
