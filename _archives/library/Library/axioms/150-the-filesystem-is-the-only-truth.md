# 150-the-filesystem-is-the-only-truth

The filesystem is the primary substrate. Plugins, themes, and UI abstractions are transient; they are merely renderers for the underlying data. To maintain sovereignty, the system must be designed to be readable and functional without any specific software layer.

## Core Principle
**Data-First, Renderer-Second.** If your system requires a specific plugin to be understood, you have lost sovereignty.
