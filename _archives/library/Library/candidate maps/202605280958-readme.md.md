# 30_Maps — Structure Notes (Maps of Meaning)

Purpose:
- High-level maps that organize clusters of Zettels.
- These are not categories; they are living documents.

Examples:
- CM-CWM Map
- Razor Governance Map
- Xi Substrate Map
- TRM Map
- RMDS Map

Rules:
- Update maps whenever a Zettel meaningfully contributes to a cluster.
- Maps evolve; they are not fixed.
