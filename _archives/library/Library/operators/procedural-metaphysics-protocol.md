# Procedural Metaphysics Protocol

This protocol defines the "Bureaucratic Sublime" as a narrative setting. It uses administrative language (forms, toner, tax codes, classification) to describe cosmic, infinite spaces, creating a sense of claustrophobic, systemic horror.

## Core Principle
**Reality is administrative.** When describing the infinite, focus on the filing, the sorting, and the compliance.

## Workflow
1. **Identify the Void:** Define the cosmic space.
2. **Apply Bureaucratic Filter:** Describe the space using the language of office management and data classification.
3. **Establish Recursive Depth:** Ensure the filing system is infinite and self-referential.
4. **Invoke Claustrophobia:** Use the scale of the archive to make the audience feel trapped by the system.
