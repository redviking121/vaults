# Six-Beat Scalable Pattern Protocol

This protocol defines the core rhythmic engine for all narrative content.

## The Six Beats
1. **Hook:** Establish the baseline.
2. **Tilt:** Introduce the syncopation (the "wrong" note).
3. **Descent:** The drift into the TRMS tension.
4. **Break:** The containment failure (the rupture).
5. **Surge:** The dialectic synthesis.
6. **Crown:** The new, coherent state.

## Core Principle
The beats are fixed; the space between them is elastic. This allows the pattern to scale from a short radio drama to a full-length series while maintaining the same "5/4" rhythmic signature.
