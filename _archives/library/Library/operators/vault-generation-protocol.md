# Vault Generation Protocol

This protocol formalizes "generational vaulting" as a standard maintenance procedure to prevent PKM entropy.

## Core Principle
Sovereignty is maintained by periodically "re-founding" the vault, migrating only the essential substrate, and pruning legacy cruft.

## Workflow
1. **Audit:** Assess the current vault's structural integrity and relevance.
2. **Migration:** Extract only the essential substrate (axioms, operators, reflections).
3. **Re-founding:** Initialize a new vault generation (vN+1).
4. **Pruning:** Leave legacy cruft in the archive.
