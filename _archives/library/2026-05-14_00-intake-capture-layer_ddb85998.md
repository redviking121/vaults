# 00_INTAKE — Capture Layer

## Purpose
- Zero-friction capture of all incoming material.
- Every paste, idea, fragment, or draft enters here.
- No editing, no linking, no thinking.

## Workflow
1. Paste content into a new file: YYYYMMDDHHMMSS.md
2. Tag with #intake if needed.
3. Move to 10_Fleeting during processing.

## Integration
- This layer acts as the **Ingestion Port** for the [[2026-05-14_xanadu-node-v0-1-mini-rfc_d980cef4]].
- Raw material captured here is the primary input for the `Task` creation lifecycle within the Xanadu runtime.
- This protocol is a specialized implementation of the [[ingestion-substrate-protocol]].
