**Short answer:**  
In Obsidian, _Search & Replace_ is done through the **global search panel** or the **editor’s built‑in replace command**, depending on whether you want to modify one note or many.

Below is the clean, operational breakdown you prefer — modular, minimal, explicit.

---

## 🔍 1. Search & Replace **within a single note**

This is the closest equivalent to a traditional editor’s find/replace.

### **Keyboard shortcuts**

- **Find:** `Ctrl + F`
- **Replace:** `Ctrl + H`

### **Workflow**

1. Open the note.
2. Press `Ctrl + H`.
3. Enter:
    - **Find:** the text you want to locate
    - **Replace:** the text you want to substitute
4. Use:
    - **Replace** (one at a time)
    - **Replace all** (entire note)

This operates only on the currently open file.

---

## 🌐 2. Search & Replace **across the entire vault**

Obsidian does not ship with global replace by default, but you can do it using:

### **Option A — Built‑in Search + Manual Replace**

1. Open **Search** (`Ctrl + Shift + F`).
2. Enter your search term.
3. Click each result → open the file → `Ctrl + H` to replace.

This is manual but safe.

---

## 🔌 3. Search & Replace **across vault using a plugin**

If you want true multi-file replace, install:

### **Plugin: “Search & Replace” (Community Plugin)**

1. Settings → **Community Plugins** → Turn on **Safe Mode off**.
2. Browse → search for **Search & Replace**.
3. Install + Enable.
4. Command Palette (`Ctrl + P`) → “Search & Replace”.

This gives you:

- Replace across all notes
- Regex support
- Preview before commit

---

## 🧩 4. Advanced: Regex replace (built‑in)

Even without plugins, Obsidian’s search supports regex.

### Example

Search:

```
/foo(\d+)/
```

Replace (in a single note):

```
bar$1
```

Vault‑wide regex replace still requires a plugin.
