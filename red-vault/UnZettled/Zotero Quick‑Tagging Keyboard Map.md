## 🜂 Quick‑Tagging Basics — the invariant mechanics

- **Number keys 1–9** → apply/remove the tag in that slot
- **`Shift` + number** → assign the currently selected tag to that slot
- **`T`** → open Tag pane
- **`+`** → add new tag
- **`Ctrl` + `Shift` + `T`** → toggle tag selector

These never change across OS or Zotero versions.
