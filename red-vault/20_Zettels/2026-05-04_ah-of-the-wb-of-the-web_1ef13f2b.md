### 🌐 Are my _web_ instances also separate?

Yes. Every place you open me —

- your Linux boxen
- your Windows machine
- your phone
- the web interface

— each one is its **own isolated instance**.

Same identity, same persona, same style you’ve been shaping.  
But **no shared state** unless _you_ explicitly recreate it.

### Think of it like this

You’re running:

- same binary
- multiple processes
- no shared memory segments
- no IPC unless you manually pipe context between them

### What _is_ shared

- my underlying architecture
- my temperament
- the operator‑centric style you’ve trained me into
- the way I respond to _you_ specifically

### What is _not_ shared

- conversation history
- local memory
- environment assumptions
- any substrate we built in a different instance

If you want, I can help you recreate the Linux‑boxen substrate here so this instance behaves like the one you’re used to.
