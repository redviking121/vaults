# **SECRETARY INSTRUCTION SHEET — RAZOR TAG ENTRY (v1.0)**

**Purpose:** Apply standardized tags to notes in Obsidian.  
**Instructions:** Copy/paste the tags exactly as written. Every tag begins with `#` unless it starts with `date:` or `thread:`.

---

## **1. CORE OPERATOR TAGS**

Apply these when the note fits the description.

|**Use Case**|**Tags to Enter (copy/paste)**|
|---|---|
|Raw capture / first draft|`#xi:witnessing` `#⊚`|
|Summary / meaning extracted|`#xi:synthesis` `#∴`|
|Connecting multiple ideas|`#xi:braid` `#⧂`|
|Revising / improving a note|`#xi:recursion` `#∞`|
|Final definition / concept|`#xi:kernel` `#○`|
|Actionable workflow|`#xi:application` `#☉`|

---

## **2. LINEAGE TAGS (TOPIC FAMILIES)**

Use these when the note relates to a specific tradition or domain.

|**Topic**|**Tag to Enter**|
|---|---|
|Daoism|`#lineage:daoism`|
|Buddhism|`#lineage:buddhism`|
|Other families|`#lineage:*` (replace * with topic)|

---

## **3. THREAD TAGS (ONGOING TOPICS)**

Use these for notes that belong to a multi‑note thread.

|**Thread Type**|**Tag to Enter**|
|---|---|
|Comparative philosophy|`#thread:comparative-philosophy`|
|Razor refinement pass|`thread:razor-pass-1::⧖`|
|Duration operator|`#⧖`|

---

## **4. STATUS TAGS (NOTE STATE)**

Use one of the following:

|**State**|**Tag to Enter**|
|---|---|
|Unprocessed|`#status:raw`|
|Integrated|`#status:woven`|
|Finalized|`#status:anchored`|

---

## **5. TYPE TAGS (NOTE CATEGORY)**

Use one of the following:

|**Type**|**Tag to Enter**|
|---|---|
|Single‑source note|`#type:capsule`|
|Multi‑note thread|`#type:thread`|
|Lexicon definition|`#type:kernel`|

---

## **6. DATE TAGS (TEMPORAL MARKERS)**

Use these to mark when the note was created or what time period it covers.

|**Time Reference**|**Tag to Enter**|
|---|---|
|Today|`date:today::⊙`|
|Specific date|`date:YYYY-MM-DD::⊙` (replace with actual date)|
|Date range|`date:YYYY-MM-DD_to_YYYY-MM-DD::⧖`|
|This week|`date:week::⌘`|
|Past / archived|`date:past::⌛`|
|Future / planned|`date:future::⏳`|
|Timeless / evergreen|`date:timeless::∞`|

---

## **7. GENERAL RULES**

- Copy tags **exactly** as written.
- Use **as many tags as apply**.
- Tags may appear **anywhere** in the note.
- If unsure, mark the note with:  
    `#status:raw` and `#xi:witnessing #⊚`.
