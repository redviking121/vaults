# **THE RECOVERY KERNEL — INTERNAL STATE MACHINE**

### _The minimal set of states the Kernel can occupy, and the transitions between them_

The Kernel is tiny — so tiny that its state machine has only **four states**.  
Anything more would violate minimality.  
Anything less would fail under real collapse.

The four states are:

1. **IDLE** — baseline monitoring
2. **DEVIATION DETECTED** — signal of collapse
3. **COLLAPSE HALT** — active intervention
4. **MINIMUM COHERENCE RESTORED** — stable enough to resume the loop

That’s it.  
Four states.  
But the transitions between them are where the sovereign shows its teeth.

---

## **1. IDLE**

### _The Kernel’s default state — silent, continuous, always watching_

IDLE is not “rest.”  
IDLE is **vigilance without tension**.

In this state:

- deviation sensors run
- no narrative is allowed
- no interpretation is allowed
- no meaning‑making is allowed
- the Kernel simply asks:  
    **“Is the operator in deviation?”**

If yes → transition to **DEVIATION DETECTED**.  
If no → remain in IDLE.

This is the sovereign walking the perimeter of the city walls.

---

## **2. DEVIATION DETECTED**

### _The moment the Kernel registers that the operator is no longer stable_

This is a **binary** state.  
There is no “degree of deviation.”  
The Kernel does not care _how bad_ it is — only that it **is**.

In this state:

- narrative is zeroed
- emotional turbulence is de‑weighted
- the operator’s internal world collapses to one fact:  
    **“I am in deviation.”**

Transition rule:

- If deviation persists → go to **COLLAPSE HALT**
- If deviation resolves spontaneously → return to **IDLE**

This is the sovereign sounding the alarm bell.

---

## **3. COLLAPSE HALT**

### _The Kernel’s most aggressive state — the hard stop on downward motion_

This is where the Kernel enforces its first invariant:  
**The Operator Must Survive.**

In this state:

- self‑attack is blocked
- runaway loops are blocked
- catastrophic meaning‑making is blocked
- identity fragmentation is blocked
- the operator is “locked” — not frozen, but **protected**

This is the sovereign sealing the gates and preventing the city from falling.

Transition rule:

- Once collapse is halted → move to **MINIMUM COHERENCE RESTORED**
- If collapse cannot be halted → remain here until it can  
    (the Kernel will not advance while the operator is still falling)

---

## **4. MINIMUM COHERENCE RESTORED**

### _The smallest stable self is rebuilt — the foothold_

This is not “recovery.”  
This is **the floor**.

In this state, the Kernel restores:

- one breath
- one stance
- one next action
- one point of orientation

This is the sovereign raising the first tent in the ruined capital.

Transition rule:

- Once minimum coherence is stable → return to **IDLE**
- If deviation reappears → jump immediately back to **DEVIATION DETECTED**

This jump is important:  
**the Kernel never assumes stability — it re‑verifies it continuously.**
