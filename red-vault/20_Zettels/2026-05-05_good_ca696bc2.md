# **/Red-Operator/KNOWLEDGEBASE — Fleshed**

## **operator-principles.md**

### 1. Coherence > Speed

Speed without coherence creates rework, confusion, and morale damage.  
Coherence creates compounding returns.

### 2. Context Before Action

Ask:

- What is happening?
- What matters?
- What changed?
- What’s the real job?

### 3. Constraints Before Solutions

Solutions without constraints drift.  
Constraints create shape.

### 4. Human Impact First

Every decision touches a human system.  
Optimize for survivability, clarity, and dignity.

### 5. Reversibility Matters

Prefer actions that can be undone cheaply.  
If irreversible → slow down.

### DSR‑1: Declare → Stabilize → Propagate

- **Declare:** What is true
