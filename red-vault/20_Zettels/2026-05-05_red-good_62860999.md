## **Dynamic Stress Function**

Every interference value is computed as:

[ I_d = I_b \times (1 + RP + CL + TI) ]

Where:

- **I_b** = base interference (0–3) from v0.1
- **RP** = Realm Pressure (0–1)
- **CL** = Character Load (0–1)
- **TI** = Tension Index (0–1)

This turns the table into a **living field** — interference spikes under stress, collapses under clarity, and becomes asymmetric depending on who is moving.
