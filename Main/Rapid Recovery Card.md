# Rapid Recovery Card

Placed in: **00-Inbox** of the Ops Vault

A one‑page, operator‑grade reference for rapid recovery from cognitive, emotional, or operational drift.

---

## Core Principle

**Restore signal first.** Action only follows once signal returns.

---

## State Machine (6 States)

### 1. Confusion — _Signal Loss_

**Signs:** Fog, looping, overwhelm.  
**Action:** Stop. Name the state.  
**Anchor:** One breath. One object. _“I am the operator.”_

### 2. Danger — _High-Risk State_

**Signs:** Panic, urgency, catastrophic thinking.  
**Action:** Freeze decisions. No communication.  
**Anchor:** Touch surface. Slow exhale. _“Stabilize first.”_

### 3. Emergency — _Rapid Reorientation_

**Signs:** Off-course but steerable.  
**Action:** 3-beat reset: Body → Breath → Boundary.  
**Anchor:** _“Not now. One thing only.”_

### 4. Normal — _Baseline Operation_

**Signs:** Thinking returns.  
**Action:** Identify next physical action.  
**Anchor:** _“Next action → ____.”_

### 5. Affluence — _Stable Breakthrough_

**Signs:** Flow, clarity.  
**Action:** Capture insights lightly.  
**Anchor:** One-sentence summary.

### 6. Power — _Self-Sustaining Momentum_

**Signs:** Clean system, low friction.  
**Action:** Protect the state.  
**Anchor:** _“Maintain integrity.”_

---

## Rapid Recovery Protocol (60 Seconds)

1. Name the state.
    
2. Stabilize the body.
    
3. Reduce scope to one action.
    
4. Re-enter the state machine.
    

---

## Mantra

**Signal first. Action second.**
