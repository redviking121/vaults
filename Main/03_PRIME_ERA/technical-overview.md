Xanadu Prime — Technical Overview
1. What Xanadu Prime Is
Xanadu Prime is a governed hypertext substrate: a system that defines how documents, links, versions, and transclusions behave at the architectural level.

Prime is not software — it is the specification, substrate, and governance layer from which software can be built.

Prime provides:

permanent addressing

deep versioning

true transclusion

bidirectional links

governed naming

explicit lineage

substrate‑level invariants

3. Core Technical Concepts
3.1 Permanent Addressing
Every surface receives a stable, permanent address that never changes, even as versions evolve.

Addresses are:

immutable

lineage‑aware

globally unique

substrate‑governed

3.2 Deep Versioning
Prime preserves every version of every surface.
No version is overwritten.
No version disappears.

Versioning is:

append‑only

lineage‑preserving

non‑retroactive

3.3 True Transclusion
Transclusion is not embedding or copying — it is referencing with integrity.

A transclusion:

points to a specific region of a specific version

is bidirectional

is lineage‑aware

cannot break

This is the core of the hypertext model.

3.4 Bidirectional Links
All links in Prime are two‑way:

A → B implies B → A

both sides know about the connection

both sides preserve the connection across versions

This eliminates link rot and orphaned references.

3.5 Governed Naming
Names are not labels — they are governed entities.

Naming is controlled by:

Naming Governance Clause

Naming Conventions

Prime Naming Registry

Names cannot drift, collide, or be reinterpreted.

3.6 Explicit Lineage
Every surface has:

an origin

a lineage chain

a propagation history

a stabilization record

Lineage is enforced by:

Lineage Statement

Chain of Custody Note

Provenance Timeline

4. Lifecycle Mechanics
Every surface in Prime follows the same lifecycle:

Declared — named and scoped

Stabilized — contradictions resolved via Razor

Propagated — enters the active corpus

Retired — only through constitutional event

This ensures deterministic evolution.
