# Muse: Raw AI Outputs Sandbox

- **Purpose**: Unedited captures from AI models (Gemini, Copilot, etc.).
- **Rules**:
  - Drop raw outputs here **immediately**.
  - **No editing**—treat as immutable logs.
  - Process: Move to `00-Inbox/` for refinement → `30 - Investigative Threads/` etc.
  - Naming: `YYYY-MM-DD-model-output.md` (e.g., `2026-03-21-gemini-stabilization.md`).
- **Why?** Prevents semantic drift; enables drift-logging.

**Example Workflow**:
1. AI → `muse/raw.md`
2. Review → `00-Inbox/processed.md`
3. Thread → `30 - Investigative Threads/drift.md`
