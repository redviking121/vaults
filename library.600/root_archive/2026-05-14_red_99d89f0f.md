# 🟥 2. **Xanadu_Data_Model**

The earliest data model we generated — the proto‑substrate before Xi/Razor.

### Core Entities

- **Xanadu Unit (XU)** — smallest addressable atom.
- **Span** — contiguous sequence of XUs.
- **Link** — typed, bidirectional, persistent relation between spans.
- **Transclusion** — a Span referencing another Span.
- **Weave** — a document as a graph of spans + links.
- **View** — a projection of a weave.
- **Trail** — a navigational path through views.

### Operator Interpretation

This was the first time we separated **weave** (the graph) from **view** (the projection).  
It was also the first time we defined **Trail** as a first‑class navigational entity.
