# 🟥 Takeaway

**Field_non_null answers the question: “Does this field contain _any_ value at all?”**  
Not valid, not typed correctly, not non‑empty — simply _not null_.

It is the second discriminator in the field‑layer, immediately after **Field_presence** and before **Field_non_empty**.

Field_non_null is the membrane that prevents **null‑value corruption**.

---

# 🟥 1. What Field_non_null _is_

Field_non_null is a **value‑level discriminator** applied only after the field is known to exist.

It evaluates:

- whether the field’s value is _not null_
- whether the field has been assigned _any_ payload
- whether the structure is safe to proceed to deeper checks

It does **not** evaluate:

- emptiness
- shape
- type correctness
- semantic validity

It is the field‑layer analogue of **Parameter_non_null** and a sub‑membrane of **Non_null_checks**.

---

# 🟥 2. Why Field_non_null matters

Null is the most dangerous “value” in the substrate because it masquerades as a value while containing **no information**.

Field_non_null prevents:

- null dereference
- null‑propagation corruption
- invalid transitions into shape or semantic checks
- silent failure modes where null is treated as meaningful

It is the membrane that ensures the field is _safe to touch_.

---

# 🟥 3. How Field_non_null is evaluated

The discriminator fires only after **Field_presence** returns true.

Then it performs a simple value‑level check:

1. Inspect the field’s value
2. Determine if it is null
3. Emit a binary result

This yields:

- **1** → field value is _not null_
- **0** → field value is null

This is the second element in the **Field_presence_vector** ordering.

# 🟥 4. Field_non_null vs. Field_non_empty vs. Field_shape

These three membranes form a strict progression:

- **Field_presence** — does the field exist
- **Field_non_null** — does the field contain _any_ value
- **Field_non_empty** — does the value contain _content_
- **Field_shape_checks** — does the value conform to expected structure

Field_non_null is the **first value‑level discriminator**.
