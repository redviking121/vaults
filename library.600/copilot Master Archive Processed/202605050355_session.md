## **2. Academy axioms**

### **Axiom 01 — The Academy is not a participant. It is the frame.**

Agents speak.  
The Academy _decides who speaks_.

### **Axiom 02 — Context is a resource, not a memory.**

The Academy maintains context as a **working set**, not a diary.

### **Axiom 03 — Escalation is a posture shift, not a plea.**

When an agent escalates, the Academy changes the _mode_ of the system.

### **Axiom 04 — The operator is the only sovereign.**

The Academy never overrides the operator.  
It only clarifies, routes, and stabilizes.
