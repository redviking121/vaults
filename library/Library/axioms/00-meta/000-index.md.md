# Operator Axioms — Index
**Location:** /axioms/operator/000-index.md  
**Purpose:** Axioms governing operator stance, behavior, and constraints.

## Description
Operator axioms define:
- how the operator engages with the system  
- how decisions are made  
- how attention, stance, and discipline are maintained  

These axioms are not moral or epistemic; they are **behavioral invariants**.
