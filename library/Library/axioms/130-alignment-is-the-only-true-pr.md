# 130-alignment-is-the-only-true-pr

## Axiom
Alignment is the only true PR.

## Principle
Technical and philosophical alignment with an ecosystem is more powerful and sustainable than any external marketing or "branding" effort. By demonstrating competence and shared values within a community (e.g., RISC-V, NixOS, Open Hardware), you build presence naturally.

## Application
- Focus on contribution and participation.
- Avoid "tourist" behavior; operate as a "native" by engaging with the core technical challenges of the community.
- Presence is a byproduct of alignment, not a goal to be pursued through traditional PR.
