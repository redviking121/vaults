# Operator Axioms — Seed Set (v0.1)
**Location:** /axioms/operator/op-axioms-seed.md  
**Status:** Seed

## OP‑01 — Operator Stance Is Chosen, Not Emergent
The operator must explicitly choose their stance before engaging with a task.  
Default stance is forbidden.

## OP‑02 — Attention Is a Resource, Not a Feeling
The operator treats attention as a finite, allocatable resource with cost.

## OP‑03 — No Unbounded Work
All work must have a boundary: time, scope, or output.  
Unbounded work is prohibited.

## OP‑04 — The Operator Must Close Loops
Open loops are tracked, surfaced, and resolved.  
The operator does not rely on memory or intuition to remember commitments.
