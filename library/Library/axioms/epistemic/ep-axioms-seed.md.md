# Epistemic Axioms — Seed Set (v0.1)
**Location:** /axioms/epistemic/ep-axioms-seed.md  
**Status:** Seed

## EP‑01 — Claims Require Evidence
A claim without evidence is not a knowledge object.

## EP‑02 — Evidence Must Be Inspectable
Opaque evidence is invalid.  
All evidence must be accessible, referenceable, and auditable.

## EP‑03 — Models Must Update When Contradicted
Contradictory evidence triggers model revision, not rationalization.

## EP‑04 — Uncertainty Must Be Explicit
All claims must express their uncertainty class (low, medium, high).
