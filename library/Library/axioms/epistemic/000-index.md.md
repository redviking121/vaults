# Epistemic Axioms — Index
**Location:** /axioms/epistemic/000-index.md  
**Purpose:** Axioms governing truth, evidence, reasoning, and belief formation.

## Description
Epistemic axioms define:
- what counts as evidence  
- how claims are evaluated  
- how uncertainty is handled  
- how models are updated  

These axioms ensure the Library remains coherent and non‑self‑contradictory.
