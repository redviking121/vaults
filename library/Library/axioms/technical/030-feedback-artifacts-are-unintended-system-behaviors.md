# 030-feedback-artifacts-are-unintended-system-behaviors

## Definition
A feedback artifact is a loop within a system that has developed the capacity to generate output—effectively "learning to speak." It is an emergent property of a system that has become sufficiently complex or recursive.

## Implications
- **Observability:** These artifacts are often mistaken for intentional system behavior, but they are usually reflections of the system's own internal noise or recursive loops.
- **Risk:** If left unmonitored, feedback artifacts can distort the system's primary objectives by creating "static" that mimics meaningful data.
- **Management:** Rather than suppressing them, they should be identified as diagnostic indicators of system state.
