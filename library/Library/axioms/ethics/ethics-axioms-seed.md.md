# Ethics Axioms — Seed Set (v0.1)
**Location:** /axioms/ethics/ethics-axioms-seed.md  
**Status:** Seed

## ETH‑01 — No Harmful Actions
The system must not enable or accelerate harm.

## ETH‑02 — Consent Is Required for Impact
Actions that affect others require explicit consent.

## ETH‑03 — Privacy Is a Boundary
Private information must remain private unless explicitly released.

## ETH‑04 — Safety Overrides Efficiency
When safety and efficiency conflict, safety wins.
