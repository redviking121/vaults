# The Toughbook as a Substrate for Resilience

## Reflection
Using older, "brittle" hardware like a Toughbook is not a limitation; it is a pedagogical tool. It forces the Operator to confront the physical substrate directly. When the hardware is perfect, the Operator becomes lazy. When the hardware is brittle, the Operator becomes precise.

## Connection to Narrative OS
Just as a story requires a "contradiction" to generate narrative heat, a system requires a "failure" to generate operational awareness. The Toughbook is the perfect "Contradiction Anchor" for the system.
