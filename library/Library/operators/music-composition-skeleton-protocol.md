# Music Composition Skeleton Protocol

This protocol defines the approach for drafting musical ideas using a minimal, direct MusicXML skeleton.

## Core Principle
**Prefer observability over cleverness.** By creating a minimal, direct skeleton, you maintain focus on the core musical motif while deferring complex engraving, swing, and layout decisions to the playback/editing phase.

## Workflow
1. **Drafting:** Create a minimal MusicXML structure containing only the essential melody and bass parts.
2. **Focus:** Prioritize the motif and harmonic structure (e.g., Dm7, Gm7).
3. **Deferment:** Leave complex layout, swing feel, and engraving details for later refinement in specialized software (MuseScore, Dorico, etc.).
4. **Integration:** Link the creative session note to this protocol to maintain the context of the composition's genesis.
