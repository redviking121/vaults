# Universal Story Pattern Protocol

This protocol defines the modular template for generating narratives within the Xi/Razor substrate.

## Core Principle
Structure is a container for creative chaos. By using a standardized, modular template, you transform the blank page into a process of filling slots.

## The Template
- **Hook:** The initial disruption.
- **Tilt:** The shift in perspective.
- **Descent:** The complication.
- **Break:** The point of no return.
- **Surge:** The climax.
- **Crown:** The resolution.

## Workflow
1. **Slotting:** Map raw ideas into the template slots.
2. **Refinement:** Iterate on the content within the fixed structure.
3. **Integration:** Archive the structured story for future retrieval and scaling.
