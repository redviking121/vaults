# Vault Topology Diagnostic Protocol

This protocol defines the method for auditing the vault's structure to ensure it remains a "knowledge reactor" rather than a "data graveyard."

## Workflow
1. **Snapshot:** Perform a high-altitude review of the current folder structure and link density.
2. **Mask Identification:** Identify which notes fall into the three core masks:
    - **Systems:** The infrastructure, protocols, and axioms.
    - **Comms:** The creative sessions, transcripts, and external outputs.
    - **Intelligence:** The synthesized reflections, maps, and emergent insights.
3. **Gap Analysis:** Identify areas where the "Masks" are bleeding into each other or where one is under-represented.
4. **Re-Calibration:** Adjust the file tree or link structure to restore the balance of the three-layer substrate.
