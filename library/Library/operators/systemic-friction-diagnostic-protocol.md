# Systemic Friction Diagnostic Protocol

This protocol formalizes the process of bypassing system-level constraints (sandboxes, Wayland/X11 conflicts, Electron limitations) to restore operational clarity.

## Workflow
1. **Identify:** Isolate the friction point (e.g., font scaling, window management, sandbox restriction).
2. **Analyze:** Map the friction to the underlying architecture (e.g., Electron + Wayland).
3. **Bypass:** Apply a forensic workaround to restore sovereignty.
4. **Document:** Record the bypass in the Archive to prevent future friction.
