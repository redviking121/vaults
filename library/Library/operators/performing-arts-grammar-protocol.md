# Performing Arts Grammar Protocol

This protocol defines the use of the Xi Razor as a timing and beat-selection engine for narrative and performance.

## Core Principle
**The Razor as a Beat-Selector.** Use the Razor’s cycle to map the emotional and logical trajectory of any performance.

## Workflow
1. **Declare:** Establish the baseline rhythm or premise.
2. **Stabilize:** Allow the audience to settle into the expected beat.
3. **Contradict:** Introduce the syncopation (the "skip" in the beat).
4. **Reconcile:** Resolve the contradiction (the punchline or dramatic turn).
5. **Propagate:** Carry the new rhythmic energy into the next phase.
