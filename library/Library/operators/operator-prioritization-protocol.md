# Operator-Grade Prioritization Protocol

This protocol defines the audit process for maintenance tasks to ensure energy is directed only at the critical path.

## The Audit
Before performing any maintenance task, evaluate it against these three criteria:
1. **Dangling Backlinks:** Does this task resolve a broken link that prevents navigation?
2. **Blocked Workflows:** Does this task unblock a process that is currently halted?
3. **Structural Drift:** Does this task prevent the system from diverging from its core architecture?

## The Decision
- If the answer to all three is **NO**: **Drop it. Move on.**
- If the answer to any is **YES**: **Execute immediately.**

## Core Principle
**Maintenance is only valuable if it is gating.**
